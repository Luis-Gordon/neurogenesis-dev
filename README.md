# neurogenesis-dev

Comments backend for [neurogenesis.dev](https://neurogenesis.dev) — powered by [Giscus](https://giscus.app) and GitHub Discussions.

## About

This repository hosts the GitHub Discussions used by the Giscus comment system on [neurogenesis.dev](https://neurogenesis.dev). Blog post comments are mapped to discussion threads by URL pathname.

The blog source code lives in a separate private repository.
